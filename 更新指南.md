打开 `AI 协作成熟度评估.html`，找到 `questions` 数组（约第 120 行开始）：

```javascript
const questions = [

            { text: "选项 A 文本", score: 1 },
            { text: "选项 B 文本", score: 2 },
```

## ⚠️ 常见问题排查

### Q1: 推送失败怎么办？
```bash
# 尝试拉取最新代码
git pull origin main

# 解决冲突后再次推送
git push
```

### Q2: 部署后网站还是旧版本？
- 清除浏览器缓存（Ctrl + Shift + Delete）
- 使用无痕模式打开
- 等待 2-3 分钟再刷新

### Q3: 不小心提交了敏感信息？
```bash
# 立即撤销最后一次提交（保留本地修改）
git reset --soft HEAD~1

# 修改文件，移除敏感信息
# 重新提交
git commit -m "修复敏感信息泄露"
git push --force
```

### Q4: 想回退到之前的版本？
```bash
# 查看提交历史
git log --oneline

# 回退到指定版本（替换 COMMIT_ID）
git reset --hard COMMIT_ID
git push --force
```

## 🎨 自定义扩展

### 添加新的评估维度
1. 在 CSS 中添加新颜色：
```css
.dim-new { background: rgba(100, 150, 200, 0.3); color: #5A7B9C; }
```

2. 在 JavaScript 中添加维度映射：
```javascript
const dims = { 
    delegation: '委托', 
    description: '描述',
    discernment: '辨别',
    diligence: '审慎',
    newdim: '新维度'  // 新增
};
```

3. 添加新题目：
```javascript
{
    dimension: "newdim",
    text: "新问题",
    options: [...]
}
```

### 修改评分逻辑
找到 `showResult()` 函数（约第 380 行），调整百分比阈值：

```javascript
if (pct < 0.3) pattern = 'explorer';      // 调低 L1 阈值
else if (pct < 0.5) pattern = 'collaborator';
else if (pct < 0.7) pattern = 'delegator';
else pattern = 'orchestrator';
```
